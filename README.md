# connexa

# Gestão Financeira Familiar com Inteligência Artificial
 
## Visão Geral do Projeto
 
**Connexa** é um projeto de Ciência de Dados I que visa transformar a gestão financeira familiar de uma tarefa burocrática em um planejamento de metas aspiracionais. Usando **Modelos de Série Temporal e Otimização**, o app Connexa não apenas rastreia gastos, mas projeta a realidade financeira e otimiza o comportamento familiar para alcançar objetivos de longo prazo (ex: viagens, reformas, grandes compras).
 
## O Problema que Connexa Resolve
 
A maioria dos apps de finanças foca no 'controle', gerando estresse e baixa adesão. O Connexa inverte essa lógica: a IA atua como um **Consultor de Metas**, respondendo à pergunta: "O que precisamos fazer hoje para viabilizar nossa viagem de 2026?".
 
## O Core de Ciência de Dados
 
O projeto acadêmico focará na construção e validação dos seguintes modelos:
 
1.  **Modelo Preditivo (Série Temporal):** Previsão do fluxo de caixa e capacidade de poupança futura, essencial para determinar a viabilidade das metas.
2.  **Modelo de Otimização/Recomendação:** Geração de planos de ação específicos e acionáveis (ex: "Reduzir Lazer em X% e Alimentação Fora em Y%").
 
## Estrutura do Repositório
 
A estrutura segue o padrão de projetos de Data Science para garantir reprodutibilidade e modularidade:
 
- `data/`: Dados brutos e processados para o treinamento do modelo.
- `notebooks/`: Análise Exploratória de Dados (EDA) e prototipagem.
- `src/`: Scripts Python finais e robustos (pipelines de dados e treinamento de modelo).
